# Notes — reading & learning

Scratch space for thoughts while reading or learning. Excluded from the Jekyll build (see `_config.yml` `exclude:` list) so this never appears on the live site, but tracked in git so it follows the repo across machines.

## Workflow

1. Add a dated section at the TOP: `## YYYY-MM-DD — topic`
2. Jot freely — quotes, links, half-thoughts, equations, open questions
3. When a section has enough substance to become a post:
   - Copy it into `_drafts/YYYY-MM-DD-title.md`
   - Refine the prose into post form
   - Move to `_posts/YYYY-MM-DD-title.md` when ready to publish
